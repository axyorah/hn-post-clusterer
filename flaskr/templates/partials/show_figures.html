Amount of posts in each cluster.

Post embeddings projected to 2D plane.

Embedding for each post is calculated by collecting all comments that belong to given post, calculating all the individual sentence embeddings for these comments and averaging the result.

BERT models that were use here produce 768- (default) or 384-dimensional (mini) embeddings vectors. The figure above shows the 2D projecttion of the embeddings that was obtained with PCA. Linear dimensionality-reduction methods like PCA are simple and quick, but the result might not always look satisfactory. If you want clusters to be more, well, clustered, you can try running nonlinear t-SNE method. Just push the Prettify button below. It will take some time...
